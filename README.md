# Çiçeksepeti React Bootcamp Graduation Project

This project made within the Çiçeksepeti react bootcamp.

## Used Technologies

-React
-React redux
-Redux thunk
-Redux devtool extension
-React dom
-Axios

## Installation

In the project directory you can run:

```console
npm install
npm start
```
